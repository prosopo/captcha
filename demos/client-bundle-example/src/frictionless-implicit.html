Procaptcha Frictionless Mode

This example demonstrates how to use Procaptcha in frictionless mode with implicit rendering.

How It Works

This example demonstrates how to use Procaptcha in frictionless mode with implicit rendering:

  1. Add the data-procaptcha attribute to your form
  2. Import and initialize Procaptcha
  3. The CAPTCHA will be automatically rendered
  4. Handle the verification result in the callback function

The key elements are:

// Import Procaptcha
import { load } from "%VITE_BUNDLE_URL%"
import { CaptchaType } from "@prosopo/types";

// Initialize Procaptcha
load({
    siteKey: "%PROSOPO_SITE_KEY%",
    captchaType: CaptchaType.frictionless
}).then(function(procaptcha) {
    // Procaptcha loaded successfully
    console.log('Procaptcha loaded');
});
