Procaptcha Invisible Slider Mode

This example demonstrates how to use Procaptcha in invisible slider mode with explicit rendering.

How It Works

This example demonstrates how to use Procaptcha in invisible slider mode with explicit rendering:

  1. Import the Procaptcha render function
  2. Create a container for the invisible CAPTCHA (usually hidden)
  3. Render the CAPTCHA explicitly with mode: 'invisible'
  4. When the user clicks the submit button, execute the CAPTCHA verification
  5. Handle the verification result in the callback function

The key elements are:

// Import the render function
import { render, execute } from "%VITE_BUNDLE_URL%"
import { CaptchaType } from "@prosopo/types";

// Render invisible CAPTCHA
const widgetId = render(document.getElementById('procaptcha-container'), {
    siteKey: "%PROSOPO_SITE_KEY%",
    captchaType: CaptchaType.slider,
    mode: 'invisible',
    callback: handleCaptchaResponse,
    "failed-callback": handleCaptchaFailed
});

// Execute when user clicks submit
document.getElementById('submit-button').addEventListener('click', function() {
    // Validate form
    if (!validateForm()) {
        return;
    }

    // Execute CAPTCHA verification
    execute();
});
