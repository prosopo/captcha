Procaptcha Slider Mode

This example demonstrates how to use Procaptcha in slider mode with implicit rendering.

How It Works

This example demonstrates how to use Procaptcha in slider mode with implicit rendering:

  1. Include the Procaptcha script in the head of your document
  2. Add a div with class procaptcha to your form
  3. Add the data-sitekey attribute with your site key
  4. Add the data-callback attribute with the name of your callback function
  5. Add the data-captcha-type attribute with the value "slider"
  6. The CAPTCHA verification happens when the user clicks the submit button

The key elements are:

<script type="module" src="%VITE_BUNDLE_URL%"></script>

<div
    class="procaptcha"
    data-sitekey="%PROSOPO_SITE_KEY_SLIDER%"
    data-failed-callback="onCaptchaFailed"
    data-callback="onCaptchaVerified"
    data-captcha-type="slider"
></div>
