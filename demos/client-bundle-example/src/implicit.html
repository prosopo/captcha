Simple Procaptcha Demo

This example demonstrates the simplest way to use Procaptcha with implicit rendering.

How It Works

This example demonstrates the simplest implementation of Procaptcha with implicit rendering:

  1. Include the Procaptcha script in the head of your document
  2. Add a container with the class "procaptcha" and a data-sitekey attribute
  3. Optionally add other data attributes for customization
  4. When the form is submitted, the procaptcha-response token will be included automatically

The key elements are:

<script type="module" src="./assets/procaptcha.bundle.js" async defer></script>

<div class="procaptcha" 
     data-sitekey="%PROSOPO_SITE_KEY%" 
     data-callback="yourCallbackFunction"
     data-theme="light">
</div>
